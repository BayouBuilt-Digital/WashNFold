# WashNFold Brand Guide

## Color Palette

### Primary Colors

| Color            | Hex       | Usage                                              |
|------------------|-----------|------------------------------------------------------|
| Dark Navy        | `#1B3A4B` | Headings, "Wash" text, primary dark text             |
| Brand Green      | `#7CC444` | "NFold" text, CTA buttons, checkmarks, accents       |
| Teal             | `#3BA0B5` | "yesterday's" italic accent, "Coming Soon" label     |

### Secondary Colors

| Color            | Hex       | Usage                                              |
|------------------|-----------|------------------------------------------------------|
| Sky Blue         | `#64B5F6` | Favicon gradient start, subtle accents               |
| Slate Gray       | `#374151` | Body paragraph text                                  |
| Medium Gray      | `#6B7280` | Secondary text, "LAUNDRY" subtitle, stat labels      |
| Light Blue-White | `#F0F7FF` | Soft background tint                                 |
| White            | `#FFFFFF` | Card backgrounds, button text, page background       |

### Gradient

| Name             | Values                    | Usage                              |
|------------------|---------------------------|--------------------------------------|
| Favicon Gradient | `#64B5F6` → `#7CC444`    | Favicon background (blue to green)   |
| Background Wash  | `#FFFFFF` → `#F0F7FF`    | Subtle page background tint          |

## Typography

### Font Family

| Font   | Weights Used          | Source                  |
|--------|-----------------------|--------------------------|
| Nunito | 300, 400, 600, 700, 800 | Google Fonts            |

### Usage

| Element              | Weight | Style   | Example                          |
|----------------------|--------|---------|-----------------------------------|
| Page Headings        | 800    | Normal  | "Laundry day just became"         |
| Accent / Emphasis    | 800    | Italic  | "yesterday's problem."            |
| Section Labels       | 600    | Normal  | "COMING SOON TO MAURICE, LA"      |
| Body Text            | 400    | Normal  | Paragraph copy                    |
| Stat Values          | 700    | Normal  | "24-48h", "$125", "4x"            |
| Stat Labels          | 600    | Normal  | "TURNAROUND", "BASIC/MO"          |
| Subtitle / Captions  | 400    | Normal  | "LAUNDRY", card feature text       |

### Letter Spacing

- **Section labels / All-caps text**: `0.12em`
- **"LAUNDRY" subtitle**: `8px`

## Logo

- **"Wash"** in Dark Navy (`#1B3A4B`)
- **"NFold"** in Brand Green (`#7CC444`)
- **"LAUNDRY"** in Medium Gray (`#6B7280`)
- **Font**: Nunito, 800 weight

## Favicon

- 64x64 rounded square (`rx="14"`)
- Background: linear gradient `#64B5F6` → `#7CC444`
- White glass bubble decorations
- White water swirl accents
- "WNF" text in white, Nunito 800

## Button Style

- **Primary CTA**: Brand Green (`#7CC444`) background, white text
- **Font**: Nunito, 700 weight
- **Shape**: Rounded
